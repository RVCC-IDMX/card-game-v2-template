# Guess My Card

## Installation

```bash
npm install
```

## VS Code Extensions

1. Prettier
2. ESLint

## Write Code

1. Card game logic in app.js

## Run development

```bash
npm run dev
```

It will be served on port 3000

[http://localhost:3000/](http://localhost:3000/)

## Run build

```bash
npm run build
```

## Run server

```bash
npm run serve
```

It will be served on port 5000

[http://localhost:5000/](http://localhost:5000/)
